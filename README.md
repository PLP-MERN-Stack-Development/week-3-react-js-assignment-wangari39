
# React Tailwind App

## Setup Instructions

1. Install dependencies:

```bash
npm install
```

2. Run the app:

```bash
npm run dev
```

## Project Structure

- Reusable components in `src/components`
- Pages in `src/pages`
- Context and utilities prepared for advanced use

## Deployment

Deploy using [Vercel](https://vercel.com) or [Netlify](https://www.netlify.com/).
